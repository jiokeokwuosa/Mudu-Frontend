# MuduPay-Frontend

## Required Features
- Authenticate 
- Create lists 
- Edit lists 
- Read all your lists 
- Mark lists as completed 
- Delete lists 


## Technologies

- React Native
- Redux

## Requirements and Installation

To install and run this project you would need to have installed:
- Node Js
- Git

To run:
```
$ git clone https://github.com/jiokeokwuosa/MuduPay-Frontend.git
$ cd MuduPay-Frontend
$ npm install
$ npm run android
```
